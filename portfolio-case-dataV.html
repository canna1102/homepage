DataV dashboard for real estate data

Orientation

Project

Abstract

In this project, we create a data visualization dashboard as a business intelligence solution for real estate data.

This solution is built based on property transaction records crawling from internet. Everyday, we obtain three types of house price, namely, new house, second-hand house and rent house. To make the price and price variance meaningful, we design several indicators, including transaction volume, price trend, average price on map, price distribution and so on.

The project is developed with DataV, a product powered by AlibabaCloud Computing Service.

Technique

Node.js, MySQL, DataV
